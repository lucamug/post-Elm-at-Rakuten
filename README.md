# Elm in Rakuten

Repository about the post **Elm in Rakuten**.

For developing, you need to have [Elm installed](https://guide.elm-lang.org/install/elm.html).

Run

```
elm reactor
```

Then access 

* http://localhost:8000/src/Illustration.elm
